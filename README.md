# DataVis App 🧠📊

This is a Flask-based web application for exploring and visualizing tabular datasets (CSV/XLSX).  
It includes features like:

- Uploading and inspecting datasets
- Auto-generated statistical summaries
- Correlation heatmaps
- Time series forecasting with lagged features
- Interactive jointplots and histograms
- An experimental chatbot-style insight assistant

---

## ⚠️ Status

**This project is a hobby/learning tool and is still under active development.**  
It is not intended for production or real-world usage. I'm using it to sharpen my skills in:

- Flask web development
- Data visualization (Matplotlib, Seaborn)
- Machine learning (basic time-series modeling)
- Web app architecture and state management

---

## 🧰 Tech Stack

- Python 3.10+
- Flask
- Pandas / Numpy
- Seaborn / Matplotlib
- Scikit-learn
- Bootstrap (via templates)
- Jinja2 templating
